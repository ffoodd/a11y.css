    set-minimum-level

    @mixin set-minimum-level($level) { ... }
    View source

    Description

    Defines the minimum level used by a11y.css. Either:

    • error for errors only;
    • warning for errors and warnings;
    • obsolete for everything but advices;
    • advice for everything.

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $level—noneString—none

    Output

    Nothing

    Example

    Defines the minimum level to error to display only errors.

    @include set-minimum-level("error");

    Throws

    • Level #{$level} is not supported. Please choose one of #{$levels}.

    Requires

    • [variable] themes
