    @mixin set-locale($language) { ... }

    Description

    Defines the language used by a11y.css. For now, only fr and en allowed.

    Parameters

    parameter Name parameter Description parameter Type parameter Default value
    $language — none String — none

    Example

    Defines the language to fr.

     @include set-locale("fr");

    Throws

    • Language #{$language} is not supported. Pull request welcome!

    set-minimum-level 

    @mixin set-minimum-level($level) { ... }
    View source

    Description

    Defines the minimum level used by a11y.css. Either:

    • error for errors only;
    • warning for errors and warnings;
    • obsolete for everything but advices;
    • advice for everything.

    Parameters

    parameter Name parameter Description parameter Type parameter Default value
    $level — none String — none

    Example

    Defines the minimum level to error to display only errors.

     @include set-minimum-level("error");

    Throws

    • Level #{$level} is not supported. Please choose one of #{$levels}.

    Requires

    • themes
